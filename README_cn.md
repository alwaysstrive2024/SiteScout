

# 🕵️‍♂️ SiteScout (灵嗅)

**自定义站点聚合搜索 + LLM 深度精炼平台**

-----

## 🌟 项目定位

**SiteScout** 是一款为“硬核搜索者”设计的智能引擎。它打破了传统搜索引擎“给什么看什么”的局限，将搜索的主动权交还给用户。

你可以把它想象成一个**“带大脑的搜索过滤器”**：它像 Google 一样简单易用，但允许你精确指定搜哪些站、看多少条、以及让哪个 AI 帮你总结。

-----

## ✨ 核心亮点

* **🖥️ 类 Google 的极简交互**：提供直观的 Web 搜索界面，零门槛上手。
* **🛠️ 自由定义的“垂直搜索池”**：用户可动态添加/删除目标站点（如 `reddit.com`, `v2ex.com`, `吾爱破解`），彻底告别垃圾广告和无关信息。
* **🎚️ 颗粒度精准控制**：
    * **保留条目 (Raw N)**：自主设定每个站点抓取的原始结果数量，确保不漏掉任何蛛丝马迹。
    * **精炼深度 (LLM K)**：自主设定提交给 AI 分析的条目权重，平衡“信息广度”与“分析速度”。
* **🧠 模型自助超市 (Model Selector)**：内置多 LLM 支持，用户可根据偏好切换 **DeepSeek-R1**（深度推理）、**GPT-4o**（通用总结）或 **Claude 3.5**（精准提取）。
* **⚡ 响应式结果矩阵**：不仅仅展示链接，更通过 LLM 自动提取资源密码、验证下载有效性并生成对比表格。

-----

## 🏗️ 平台架构

1.  **前端界面 (Frontend)**：基于 React/Next.js 构建，支持搜索参数的动态配置（Slider 拖动条选择 N/K 值）。
2.  **调度中枢 (Orchestrator)**：基于 Python `asyncio` 异步框架，同时并发请求多个站点。
3.  **智能层 (Intelligence Layer)**：支持通过 API 接入多种大模型，执行 Map-Reduce 任务，将海量原始数据压缩为高价值情报。
4.  **适配层 (Adapter)**：标准化对接 Bocha AI、Exa AI、Serper 等专业搜索接口。

-----

## 📸 功能演示 (拟)

**用户在界面输入：**
* **搜索词**：`2026 最新大语言模型评测报告`
* **指定站点**：`arxiv.org`, `github.com`, `medium.com`
* **参数设置**：每个站取 $15$ 条结果，其中前 $5$ 条由 **DeepSeek-V3** 总结。

**系统产出：**
1.  **AI 综合对比表**：对比三个站点的核心观点，提取关键指标。
2.  **源站穿透列表**：展示 45 条去重后的原始链接。

-----

## 🛠️ 技术栈

* **前端**：React / Tailwind CSS (打造极简、流畅的 UI/UX)
* **后端**：FastAPI / Python (高并发异步处理)
* **大模型**：DeepSeek, OpenAI, Anthropic (提供多模型选择)
* **搜索技术**：Bocha AI / Exa AI (专业的结构化数据搜索)

-----

## 🤝 愿景

**SiteScout** 致力于消除互联网的信息熵。我们不生产内容，我们只做互联网内容的“精炼厂”，让每一个搜索请求都能直达价值核心。
